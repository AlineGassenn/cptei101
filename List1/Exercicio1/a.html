HTML5


HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet.



Desenvolvimento Web


Desenvolvimento web é o termo utilizado para descrever o desenvolvimento e construção de aplicações para a web, com a finalidade de se obter um conjunto de funcionalidades para atender uma necessidade.



CCS3


CSS3 é a terceira versão de um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.
